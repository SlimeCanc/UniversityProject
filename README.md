# Проект для университета плешка название MovieGO

Создан для поиска фильмов которые были добавлены в ручную

Чтобы запустить сайт нужно скачать все плагины из requirements.txt или написать команду pip install -r requirements.txt затем надо сделать виртуальное пространство и запустьть сайт через run.py




В помощи использовался GigaChat от сбера
